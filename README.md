# 消息互传工具
当我们在测试的时候，拿到一个新手机，不想要登录微信或qq，或者手机没有安装微信或qq，需要发送消息或者发送文件到另一个手机，此时可以使用该网址，进行传消息，省去很多麻烦事情；





## 功能
- 拖拽上传：支持拖拽图片发送，直接拖拽到消息列表展示区域（上传支持进度展示）
- 截图发送：支持截图之后，在输入框区域直接粘贴自动发送图片到消息列表
- 当前在线：当前在线人数展示，支持上线通知



**说明：** 

* 没有做任何的数据保存
* my-chart 是前端页面
* server 是后台客户端
